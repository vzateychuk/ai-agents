# No Emoji Policy

**STRICT RULE** with NO exceptions. NEVER use emoji, Unicode symbols, or any graphical characters (like checkmarks, crosses, stars, flags, warning signs, etc.) in ANY documents, code comments, or documentation files.

## Instead of emoji, use:

- Plain text status indicators: "RESOLVED", "PARTIALLY RESOLVED", "PENDING", "CRITICAL", "IMPORTANT"
- "-" or "*" for lists, never emoji
- Plain text headers without any symbols
- Text labels like "[RESOLVED]" or "[PENDING]" instead of emoji

## When editing existing content

If you see emoji in existing documents, remove them and replace with the plain text equivalents above.
